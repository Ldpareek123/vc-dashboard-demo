📝 How this works

1. The Job Card Register assigns work to operator codes (EMP011–EMP015 — Production Supervisors / machine operators).
2. Each operator code is mapped to the corresponding employee in the 21-person Appraisal Scorecard.
3. Productivity (avg. job completion %) and Quality (linked NCR count) are auto-calculated per employee — the same evidence used in the Excel workbook.
4. Supervisors enter the final 1–5 KPI scores in the Excel/Word scorecards — these figures are supporting evidence, not the final score.
